## 中文

### 简介

Aromatic 是一个开源的多模型协作辩论平台，旨在探测和展示大语言模型（LLM）在复杂判断中的**认知芳香性（Model Aromaticity）**。

- **无需提示词赋予身份**：模型自带稳定的认知倾向，在辩论中自然形成分歧。
- **多模型交叉审视**：支持最多 6 个模型同时辩论，互相质疑、修正、联合决策。
- **双语界面与提示词**：一键切换中英文，UI 与提示词模板同步更新。
- **跨平台可复现实验**：提供便携免安装包，任何无开发环境的电脑均可运行。

### 核心特性

- **三种辩论模式**：AA 单 Agent 辩论、二元对立 (2+N)、六元分发 (3-6 模型)
- **本地账户与模型串管理**：支持多账户、多模型串，方便切换不同 API 配置
- **对话历史库**：自动保存每次辩论记录，支持导出 Markdown/TXT
- **数据包扩展**：通过数据包自定义提示词模板和辩论流程
- **迭代次数控制**：支持 0-5 轮交叉审视，灵活控制实验深度

### 快速开始

1. 前往 [Releases](https://github.com/XCXC000/Aromatic/releases) 下载最新版本的 `Aromatic-v2-windows-x64.zip`
2. 解压到任意目录，双击 `aromatic.exe` 运行
3. 在设置中配置你的 API 账户和模型串
4. 返回主页，输入问题，选择模型，开始辩论

### 环境要求

- Windows 10 或更高版本（64 位）
- 无需安装 Flutter、Dart 或任何开发工具

## English

### Introduction

Aromatic is an open-source multi-model collaborative debate platform designed to probe and demonstrate **Model Aromaticity** — the stable cognitive tendencies exhibited by large language models (LLMs) in complex judgments.

- **No prompt-engineering for roles**: Models bring their own inherent cognitive biases, forming natural divergence in debates.
- **Multi-model cross-examination**: Up to 6 models debate simultaneously, questioning, revising, and making joint decisions.
- **Bilingual UI & prompts**: Switch between Chinese and English with one click; all interface texts and prompt templates update accordingly.
- **Reproducible experiments anywhere**: Portable executable package runs on any Windows machine without developer tools.

### Key Features

- **Three debate modes**: Single Agent Debate (AA), Duel (2+N), Hexad Distribution (3-6 models)
- **Local account & model chain management**: Manage multiple accounts and model chains for flexible API configurations
- **Conversation library**: Auto-saves every debate session, supports Markdown/TXT export
- **Data pack extensibility**: Customize prompt templates and debate workflows via data packs
- **Adjustable iteration depth**: 0-5 rounds of cross-examination for fine-grained experimental control

### Quick Start

1. Go to [Releases](https://github.com/XCXC000/Aromatic/releases) and download the latest `Aromatic-v2-windows-x64.zip`
2. Extract the archive and double-click `aromatic.exe`
3. Configure your API accounts and model chains in Settings
4. Return to the main screen, enter a question, select models, and start the debate

### Requirements

- Windows 10 or later (64-bit)
- No Flutter, Dart, or other development tools required
